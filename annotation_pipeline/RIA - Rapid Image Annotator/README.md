\#Rapid Image Annotator



This folder contains files associated with version 2.6.10 of the Rapid Image Annotator



\## Feature additions

&nbsp;   - Made filename display generic to tasks

&nbsp;   - Made text displaying annotation times more readable

&nbsp;   - Improved box selection and deselection

&nbsp;   - General aesthetic improvements



\## Bug fixes



&nbsp;   - Made box comments less likely to fall outside image 

&nbsp;   - Prevented binding of mouse buttons to simultaneous image navigation and comments

&nbsp;   - Allowed saving of zero width / height boxes

&nbsp;   - Prevented pagination from accidently overwriting annotations

&nbsp;   - Eliminated edge case where new boxes could overwrite old box data

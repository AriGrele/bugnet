\#Web scraping

\##Input files
All files expect one item per line. Taxon names are case insensitive. License and authentification items are case sensitive.

&nbsp;\* order\_filter - GBIF search will only include taxa in these orders
 \*family\_filter - GBIF search will exclude taxa in these families
 \*license\_filter - GBIF search will only include images with these licenses
 \*gbif\_auth - GBIF username, password, and email, in that order

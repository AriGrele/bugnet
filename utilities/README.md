# Utility scripts for deployment

## scripts

 * bugnet.py is the main script for deploying the bugnet localization and classification pipeline
 * enet_models contains classes used by the EfficientNet classification models
 * GBIF.py contains code for interacting with GBIF via the pygbif library
 * functions contains helper functions used across scripts

## Folders
Functions, models, and utils contain scripts from YOLOv5 and are required for running localization models
